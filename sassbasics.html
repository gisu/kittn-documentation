Coding Tip

Class names that are generated by "Kittn", follow the BEM syntax. BEM is one of the easiest ways to write Oriented CSS object. Surely you can build a lot more delicate with Smacss, but should not forget that the issue is simply a principle of object-oriented programming here. One should therefore not make a religion out of the topic - the final product will always be CSS with all its drawbacks.

The Develeloper is free to decide whether he wants his class named after the BEM build syntax, in which case you can rely on the BEM mixins, thus obtaining a central node to make global modifications to the syntax.

By default BEM elements marked with a double underscore __ and BEM Modifier with a double hyphen --. Who does not agree, this may change the Sass Setup file (src/sass/_setup.scss) on the by adjusting the $kittn-bem Map accordingly.

Sass Basics

The centerpiece of the toolkit consists of the Sass framework, the toolkit was built more around to replace the missing functions.

You not need to worry about CSS3 prefixes the gulp-autoprefixer will insert it on any Compile.

Structure

In the directory src/sat/ is the Framework:

sass/
.. framework/ Framework mixins and functions
.... helper/ Helpers that need to included first
.... modules/ Mixins, functions and generators
.... partials/ Predefined snippets
.. maps/ Extended Data Maps
.. styles/ Possible place for your own partials
.. _loader.scss Loading the files (the order is predetermined)
.. _setup.scss The entire configuration
.. _workfile.scss The Workfile
.. style.scss Build the Stylesheet
.. style-ie8.scss Build the Stylesheet for IE8

The name for the files style.scss may differ if you have specified a different name when you activate the toolkit, as it may be that there is no xxxxi8.scss file exists when the IE8 compatibility was turned off.

Workfile

Work always with _workfile.scss file and the contents of this file into the respective style.scss imported and converted by the compiler to a CSS file. We simply open one off the style.scss files:

This file will import the content from the _workfile, but filter out this specific content. For example, Contents that are tagged with the variable old, will not include here.

Using Jacket the developers to work on a file, route-specific styles on different files.

In Kittn you work principally with two Jacket variables. old marked contents which migrate in IE8 stylesheet and modern is intended only for modern browsers. If you want you can of course still expand.

Loader

src/sass/_loader.scss

With the _loader.scss file, all required files are loaded. The order is already given proof should not be changed.

Setup

src/sass/_setup.scss

Almost all settings can be changed and extended over _setup.scss.

$kittn-global-spacing

Global spacing unit, used for Spacer and Lines Mixins.

Number
$kittn-global-spacing: 20;

$kittn-default-unit

Default unit for unitless values. Default px

*
$kittn-default-unit: px;

$kittn-bem

BEM Prefixes for Elements and Modifiers

  • {*} element - Prefix
  • {*} modifier - Prefix
$kittn-bem: (
  element: '__',
  modifier: '--'
);

$kittn-directorys

Kittn Directory Organisation - when you change the structure from dist/ you need to change here also.

  • {*} fonts - Directory
  • {*} images - Base CSS image folder
  • {*} single - Folder from the single images
  • {*} texture - Folder from the texture files
  • {*} svg - Folder from the svg images with png fallbacks
  • {*} svgonly - Folder SVG only files
$kittn-directorys: (
  fonts: '/assets/fonts/',
  images: '/assets/img/',
  single: '/assets/img/single/',
  texture: '/assets/img/textures/',
  svg: '/assets/img/svg/',
  svgonly: '/assets/img/svgonly/'
);

$kittn-activate

Activate Base Features

  • {Bool} debugging - Activate the CSS Debugging Features
  • {Bool} box-sizing - Activate the new old Box Model ;P
  • {Bool} baseline - Activate Baseline Features
  • {Bool} hyphens - All Words would be breaked with Hyphens
  • {Bool} viewport-fix - Activate Viewport-Fixes for Windows Mobile 8
  • {Bool} mobile-first - Grid and Breakpoints oriented from Smallest to Biggest Size
  • {Bool} ie8-fallback - Integrate some IE8 Fallbacks
$kittn-activate: (
  debugging: false,
  box-sizing: true,
  baseline: false,
  antialias: true,
  hyphens: false,
  viewport-fix: false,
  mobile-first: true,
  ie8-fallbacks: false
);

$kittn-sprite

Sprite Configuration

  • {*|Bool} name - Sprite Name
  • {Bool} retina-capable - Integrate an additional Retina Sprite
$kittn-sprite:(
  name: false,
  retina-capable: true,
);

$kittn-iconfont

Iconfont Configuration

  • {*|Bool} name - Iconfont Name with 'false' deactivate the Iconfont
  • {*} prefix - Icon Prefix
$kittn-iconfont: (
  name: false,
  prefix: 'f-icon'
);

$kittn-breakpoint-map

Map off all Breakpoints and Behaviors

  • {*} [name] - Breakpoint Name
    • {Number} size - The width that activate the Trigger
    • {Bool} visibility - Set it on true to activate the visibility classes
    • {Bool} step - Set the threshold for the grid-stepper mixin
    • {Bool|Number} fontsize - Rewrite the Basefontsize for this breakpoint
$kittn-breakpoint-map: (
  1: (
    size: 320px,
    visibility: true,
    step: true,
    fontsize: false
  )
);

$kittn-grid-setup

Setup the Base Grid

  • {Bool} activate - Activates the Generator
  • {Book} silent - Generated classes are silent [%]
  • {*} container-name - Name of the container class
  • {*} row-name - Name of the row class
  • {*} colum-name - Name of the colums
  • {*} size-name - Name of the size classes [prefix]
  • {*} offset-name - Name of the offset classes [prefix]
  • {*} pushpull-name - Name of the pushpull classes [prefix]
  • {*} breakpoint - This breakpoint will modifiy the grid [$kittn-breakpoint-map]
  • {Number} colums - Maximum colums
  • {Number} spacing - Padding of the colums
  • {*} layouttype - 'inline', 'table', 'flex', 'block'
  • {*} valign - 'top', 'bottom', 'baseline', 'middle'
  • {Bool} container-adaptive - adaptive or fluid
  • {List} container-width - Main Width off the Grid
  • {*} container-position - 'center', 'left', 'right'
  • {Number} container-offset - The Size of the Outer Gutter
  • {Bool|Number} container-ie8-style - When is set on 'true' IE8 get a static Container
  • {Bool} colum-extras - Want to include the additional colum classes
  • {Bool|*} size-extend - Size classes can be extended with other classes
  • {Bool|List} size-select - You can select the size classes
  • {Bool} offset-activate - Activate the offset classes
  • {Bool} pushpull-activate - Activate the push&pull classes
$kittn-grid-setup: (
  activate: false,
  silent: true,
  container-name: 'container',
  row-name: 'row',
  colum-name: 'column',
  size-name: 'width',
  offset-name: 'offset',
  pushpull-name: 'pushpull',
  breakpoint: 4,
  colums: 12,
  spacing: 10,
  layouttype: block,
  valign: top,
  container-adaptive: false,
  container-width: 320 1180,
  container-position: center,
  container-offset: 0,
  container-ie8-style: false,
  colum-extras: false,
  size-extend: false,
  size-select: false,
  offset-activate: false,
  pushpull-activate: false
);

$kittn-fontface-map

Map off all Breakpoints and Behaviors

  • {*} [fontname]
    • {Bool|*} filename - Filename, when 'false' the Fontname will be used
    • {*} weight - The associated Fontweight
    • {*} style - The associated Font Style
    • {Bool} svgoption - Activate the addition route to the SVG Font
$kittn-fontface-map: (
  kittn: (
    filename: false,
    weight: normal,
    style: normal,
    svgoption: true
  )
);

$kittn-typo

Base Typo Setup

  • {*} unit - 'px' or 'rem'
  • {Number} base-scale - Fontsize on <html>
  • {Number} base-size - Base Fontsize in 'px'
  • {Number} line-height - Base Line-Height
$kittn-typo: (
  unit: rem,
  base-scale: 85,
  base-size: 16,
  line-height: 21
);

$kittn-active-responsive-fonts

Set the Start or End Fontsize for the responsive fontsizes. 'false' will deactivate the responsive fontsizes.

Number
$kittn-active-responsive-fonts: 85;

$kittn-typo-margin

Default Margin for Typographic Elements (set on '0' to deactivate)

  • {Number} top - Top Margin Size
  • {Number} botton - Bottom Margin Size
$kittn-typo-margin: (
  top: s(1.25),
  bottom: s(1)
);

$kittn-typo-style

Base Typo and Font Styling

  • {*} base-font - Base Font
  • {*|Bool} base-weight - Base Fontweight, 'false' will deactivate
  • {Bool|*} heading-font - Font for Headlines, 'false' will use the Basefont
  • {Bool|*} bold-font - 'false' will use the Basefont
  • {Bool|*} bold-weight - Set the Weight for Bold Font
  • {Bool|*} italic-font - 'false' will use the Basefont
  • {Bool|*} italic-style - Set the Fontstyle for italic typos
  • {Bool} link-decoration - De/Activate the Text Decoration for links
  • {Bool} link-decoration-hover - De/Activate the Text Decoration for links hovered state
  • {Bool|*} quote-font - 'false' will use the Basefont
  • {Bool|*} quote-weight - Set the Fontweight for Blockquotes
  • {Bool|*} quote-style - Set the Fontstyle for Blockquotes
  • {Bool|*} code-font - 'false' will use the Basefont
  • {Bool|*} code-weight - Set the Fontweight for Code Elements
$kittn-typo-style: (
  base-font: ff('helvetica'),
  base-weight: false,
  heading-font: ff('helvetica','Montserrat'),
  bold-font: false,
  bold-weight: bold,
  italic-font: false,
  italic-style: italic,
  link-decoration: false,
  link-decoration-hover: false,
  quote-font: false,
  quote-weight: false,
  quote-style: false,
  code-font: ff('monospace'),
  code-weight: normal
);

$kittn-typo-sizes

Base Typo Sizes

  • {Bool|*} list-fontsize - Fontsize for List Elements, 'false' use Basefontsize
  • {Bool|*} list-lineheight - Lineheight for List Elements, 'false' use Base Line-Height
  • {Number} list-margin - Sidemargin
  • {*} list-style - Values : 'disc', 'square', 'circle'
  • {Bool|Number} quote-fontsize - Fontsize for Blockquotes, 'false' use the Base Fontsize
  • {Bool|Number} quote-lineheight - Lineheight for Blockquotes, 'false' use Base Line-Height
  • {Bool|Number} cite-fontsize - Fontsize for Cite Elements, , 'false' use Basefontsize
  • {Bool|Number} cite-lineheight - Line-Height for Cite Elements
  • {Bool|Number} code-fontsize - Fontsize Code Elements
  • {Bool|Number} code-lineheight - Lineheight for Code Elements
$kittn-typo-sizes: (
  list-fontsize: false,
  list-lineheight: map-get($kittn-typo, line-height) + 4,
  list-margin: s(1),
  list-style: disc,
  quote-fontsize: map-get($kittn-typo, base-size) + 2,
  quote-lineheight: map-get($kittn-typo, line-height) + 6,
  cite-fontsize: map-get($kittn-typo, base-size) - 1,
  cite-lineheight: map-get($kittn-typo, line-height) + 0,
  code-fontsize: false,
  code-lineheight: false
);

$kittn-global-heading

Global Heading Stylings

  • {Bool|*} lineheight - 'false' or 'd' (for line-height:1) or a unitless number
  • {Bool|Number} marginbottom - 'false' or 'number'
  • {Bool|*} fontweight - 'false' or 'number'
$kittn-global-heading: (
  lineheight: false,
  marginbottom: 0,
  fontweight: 400
);

$kittn-typostyles-map

Map all possible Fontstyles

  • {*} [stylename] - Fontstyle Name
    • {Number} font-size - Fontsize in 'px' relation (will be converted)
    • {Bool|Number|*} line-height - Lineheight, Values: 'd': line-height = 1, 'c': sass calculated or 'number' or 'bool'
    • {Bool|Number} margin-bottom - Bottom Margin
    • {Bool|*} font-weight - Font Weight
    • {Bool} tag - When active it will generate also a Tag, 'false' only classes
    • {*} extend - When you want to Extend the Style you can enter here the Extend Classname like '%myclass'
$kittn-typostyles-map: (
  h1: (
    font-size: 36,
    line-height: d,
    margin-bottom: s(0.5),
    font-weight: false,
    tag: true,
    extend: '%heading'
  )
);

$kittn-generate-typostyles

$kittn-color-map

Color Definition Map. This is a collection from all used Colors. Use only use HEX Colors, when you enter a Opacity Value (between 0 to 1), Sass convert it to RGBA with a Fallback for old Browsers.

  • {*} [colorstyle] - Name for the Colorstyle, quoted
    • {Color} color - Color (HEX)
    • {Bool|Number} opacity - Opacity Value (0.0 - 1.0)
    • {*} type - Type of the Generated classes. Values: 'bc' background & textcolor, 'c' textcolor, 'b' background-color
$kittn-color-map: (
  'primary': (
    color: #23baf2,
    opacity: false,
    type: bc
  ),
  'primary-dark': (
    color: #188cb8,
    opacity: false,
    type: bc
  ),
);

$kittn-color-generator

Generator Settings for the Color Map

  • {Bool} generate - Activate the Generator
  • {*} type - 'class' or 'silent'
$kittn-color-generator: (
  generate: true,
  type: silent
);

$kittn-color-route

Route the Colors to the Elements

  • {Color} body - Body Background
  • {Color} text - Base Text Color
  • {Color|Bool} headline - Headline Color
  • {Color|Bool} blockquote - Textcolor for a Blockquote
  • {Color|Bool} highlight-bg - Background for Mark
  • {Color|Bool} hightlight-text - Textcolor for Mark
  • {Color|Bool} link - Textcolor Link Elements
  • {Color|Bool} link-hover - Textcolor for Link Elements (:hover)
  • {Color|Bool} link-active - Textcolor for Link Elements (:active)
$kittn-color-route: (
  body: cp('body'),
  text: cp('black'),
  headline: cp('gray'),
  blockquote: cp('gray'),
  highlight-bg: lighten(cp('primary'),25),
  hightlight-text: darken(cp('primary'),10),
  link: cp('primary'),
  link-hover: darken(cp('primary'),15),
  link-active: darken(cp('primary'),25),
);

$kittn-tables

Base table styling

  • {Color} background - Tablecell Backgroud
  • {Color} background-zebra - Tablecell Background Variation for Zebrastyle
  • {Color} background-hover - Tablecell Background when hovered
  • {Color} border-color - Border Color
  • {Number} margin - Margin Bottom
  • {Number} padding - Tablecell Padding
  • {Number} trigger - Trigger Size for responsive Tables
$kittn-tables: (
  background: transparent, // {Color}
  background-zebra: tint(cp(primary),20), // {Color}
  background-hover: lighten(cp(primary),20), // {Color}
  border-color: cp('gray'), // {Color}
  margin: 20px, // {Number}
  padding: 8px, // {Number}
  trigger: 400px, // {Number} Trigger Size for responsive Tables
);

$kittn-button

Button Map. You can add more than one button, any attribute is optional

  • {*} buttonname - Name of the Button
    • global
      • {Color|Bool} color - Text Color
      • {Color|Bool} background - Background Color
      • {*|Bool} display (inline-block) - Display Type
      • {Number|Bool} width - Width of the button
      • {Number|Bool} fontsize - Text Fontsize
      • {Number|Bool} lineheight - Text Lineheight
      • {Number|Bool} height - Button height
      • {*|Bool} align (center) - Text Align
      • {Number|Bool} padding-v - Vertical Padding
      • {Number|Bool} padding-h - Horizontal Padding
      • {*|Bool} weight - Font Weight
      • {*|Bool} style - Font style
      • {*|Bool} family - Font Family
      • {*|Bool} transition - State transition
      • {*|Bool} shadow - Box Shadow
      • {*|Bool} border - Border Style
      • {Number|Bool} radius - Border Radius
    • hover - hover style (optional)
      • {Color|Bool} color - Text Color
      • {Color|Bool} background - Background Color
      • {*|Bool} weight - Font Weight
      • {*|Bool} shadow - Box Shadow
      • {*|Bool} border - Border Style
    • active - Active State style (optional)
      • {Color|Bool} color - Text Color
      • {Color|Bool} background - Background Color
      • {*|Bool} weight - Font Weight
      • {*|Bool} shadow - Box Shadow
      • {*|Bool} border - Border Style
$kittn-button: (
  default: (
    global: (
      color: cp('white'),
      background: cp('primary'),
      fontsize: 14,
      lineheight: 30,
      padding-v: 20px,
      transition: 300ms ease-in-out,
    ),
    hover: (
      color: cp('white'),
      background: darken(cp('primary'),15),
    )
  )
);

$kittn-font-icons

src/sass/maps/_iconfontlist.scss

All icons from a IconFont be listed in this map - the classes would be generated with the Iconfont mixins.

Who created his Iconfonts with Icomoon, a CSS file get delivered. With some Find & Replace can this CSS file easily convert back into a map.

  • {*} iconname
  • {*} code - CSS Content Code
$kittn-font-icons: (
  check: '\f00c',
  times: '\f00d',
  home: '\f015',
  lock: '\f023',
);
